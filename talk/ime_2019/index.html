Toward an explainable Machine Learning model for claim frequency.

Abstract

In this paper, we propose an interpretable Machine Learning approach to model the claim frequency of a telematics car data set. In insurance, there have been many attempts to integrate Machine Learning models in the pricing process. But the impressive performances of these methods are often synonym of poor capacity to explain the decision mechanism. Hence, we first present a method that build a surrogate tree to explain globally an Additive Tree Model (ATM) like Random Forest or XGBoost that can achieve high accuracy predictions. Then we compare two ways to build this surrogate, with a fixed number of regions in input or using a Bayesian model selection approach to select automatically the model complexity. Then we illustrate this on a telematics data set collected for an experiment on a Pay As You Drive (PAYD) product from a Belgian portfolio. Furthermore, we compare our results to a Generalized Additive Model (GAM). Hence, we are able to evaluate the benefits of a more complex model in contrast to the performance gains.
